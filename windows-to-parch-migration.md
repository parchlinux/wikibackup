# مهاجرت از ویندوز به پارچ لینوکس: راهنمایی گام‌به‌گام برای کاربران تازه‌کار

اگر تا حالا با ویندوز کار کردی و حالا می‌خوای به سمت گنو/لینوکس بری و پارچ لینوکس انتخابته، این راهنما کمکت می‌کنه تا همه چیز رو آرام و بدون مشکل پیش ببری. پارچ لینوکس یک توزیع لینوکسی است که بر پایه آرچ لینوکس ساخته شده و با تمرکز بر سادگی و کاربرپسندی، محیط‌های میزکار قدرتمندی مثل KDE Plasma و GNOME ارائه می‌ده. این توزیع برای کاربران تازه‌کار ایده‌آله چون نصب‌کننده گرافیکی Calamares داره که فرآیند نصب رو خیلی راحت می‌کنه. علاوه بر این، پارچ از مخازن آرچ استفاده می‌کنه، پس دسترسی به هزاران بسته نرم‌افزاری داری. این راهنما برای کسانی نوشته شده که هیچ تجربه‌ای با گنو/لینوکس ندارن، پس همه چیز ساده و قدم‌به‌قدم توضیح داده شده. ما پنج گام اصلی داریم: **برنامه‌ریزی، امتحان کردن، آماده‌سازی، نصب و نگهداری**. هر گام رو با جزئیات بیشتری می‌گیم تا بتونی دنبال کنی و درک بهتری داشته باشی. لینک‌های مرتبط از منابع اصلی مثل ویکی پارچ لینوکس هم اضافه شدن تا اگر خواستی بیشتر مطالعه کنی. همچنین، در هر بخش نکات ایمنی و هشدارها رو اضافه کردیم تا از اشتباهات رایج جلوگیری بشه.

## گام اول: برنامه‌ریزی و چک کردن نیازهایت

قبل از اینکه چیزی رو تغییر بدی، باید بدونی چی لازم داری. این گام مثل اینه که قبل از سفر، چمدونت رو چک کنی و مطمئن شی همه چیزهایی که نیاز داری همراهته. اگر این مرحله رو جدی نگیری، ممکنه بعداً با مشکلات غیرمنتظره روبرو بشی، مثل عدم سازگاری سخت‌افزار یا نبود جایگزین مناسب برای نرم‌افزارهایت.

اول از همه، لیست کاملی از نرم‌افزارهایی که در ویندوز استفاده می‌کنی تهیه کن. این لیست بهت کمک می‌کنه ببینی آیا جایگزین‌های مناسبی در لینوکس وجود داره یا نه. مثلاً:

- اگر کار اداری می‌کنی: **مایکروسافت آفیس** (Word, Excel, PowerPoint) → جایگزین: **LibreOffice** که شامل Writer برای نوشتاری، Calc برای صفحه‌گسترده، و Impress برای ارائه‌هاست. LibreOffice کاملاً رایگان و سازگار با فایل‌های آفیس هست.
- اگر مرورگر کروم استفاده می‌کنی: تو پارچ **فایرفاکس** به صورت پیش‌فرض نصبه که سریع و امن هست، اما اگر کروم رو ترجیح می‌دی، می‌تونی از طریق pacman یا AUR نصبش کنی (مثلاً با دستور `sudo pacman -S chromium` برای نسخه اپن‌سورس).
- برای پخش ویدیو و موسیقی: **VLC** بهترین گزینه است که تقریباً همه فرمت‌ها رو پشتیبانی می‌کنه و در پارچ به راحتی نصب می‌شه.
- اگر برنامه‌نویسی می‌کنی: **Visual Studio Code** در پارچ موجوده و می‌تونی با `sudo pacman -S code` نصبش کنی. همچنین، برای زبان‌های مختلف مثل Python، Git و ابزارهای دیگه به راحتی در دسترسه.
- برای ویرایش گرافیک: به‌جای Adobe Photoshop، می‌تونی از **GIMP** استفاده کنی که ابزارهای پیشرفته‌ای برای ویرایش عکس داره، یا Krita برای نقاشی دیجیتال.
- اگر گیمینگ می‌کنی: Steam در لینوکس پشتیبانی می‌شه و با Proton می‌تونی بازی‌های ویندوزی رو اجرا کنی.
- برای ویرایش ویدیو: به‌جای Adobe Premiere، از DaVinci Resolve یا Kdenlive استفاده کن.

> برای مقایسه جایگزین‌ها و پیدا کردن گزینه‌های بیشتر، می‌تونی از این منبع کمک بگیری: [Windows Alternatives for Linux](https://www.tecmint.com/windows-alternatives-for-linux/). همچنین، سایت‌هایی مثل AlternativeTo.net رو چک کن تا نظر کاربران رو ببینی.

سخت‌افزارت رو هم با دقت چک کن. کارت گرافیک انویدیا یا AMD داری؟ در پارچ، ابزار **Parch Driver Manager** بهت کمک می‌کنه درایورهای مناسب رو به صورت گرافیکی نصب کنی، بدون نیاز به دستورات پیچیده. این ابزار از git.parchlinux.com در دسترسه:  
[لینک ابزار درایور](https://git.parchlinux.com/mehrad/Parch-Driver-Manager). برای کارت‌های انویدیا، ممکنه نیاز به درایورهای proprietary باشه، اما پارچ این رو ساده کرده.

پرینترها معمولاً با سیستم **CUPS** (Common Unix Printing System) پشتیبانی می‌شن که به صورت خودکار در پارچ نصبه. وب‌کم، بلوتوث و وای‌فای رو هم بهتره تست کنی – مثلاً چک کن آیا چیپست وای‌فای Broadcom یا Realtek داری، چون گاهی نیاز به درایور اضافی دارن.

اگر لپ‌تاپ داری، چک کن آیا باتری، تاچ‌پد و حسگرهای نور/صدا به درستی کار می‌کنن. سایت‌هایی مثل linux-hardware.org رو سرچ کن تا تجربیات دیگران رو ببینی.

> **نکته:** همیشه قبل از هر تغییر، از فایل‌هایت نسخه پشتیبان بگیر. بهترین روش داشتن دو نسخه پشتیبان جداست، یکی روی هارد اکسترنال (با ابزارهایی مثل File History در ویندوز) و یکی در فضای ابری مثل Google Drive یا OneDrive. اگر فایل‌های مهمی داری، از نرم‌افزارهایی مثل Acronis True Image برای بک‌آپ کامل سیستم استفاده کن. این کار جلوی از دست رفتن داده‌ها رو می‌گیره اگر چیزی اشتباه پیش بره.

## گام دوم: امتحان کردن پارچ لینوکس بدون تغییر سیستم

حالا وقتشه پارچ رو امتحان کنی بدون اینکه ویندوز رو حذف کنی یا تغییری در سیستم اصلی بدی. این گام مثل تست درایو یک ماشین جدیده – می‌تونی ببینی آیا خوشت می‌آد یا نه، بدون تعهد بلندمدت.

برای این کار، اول یک فلش درایو حداقل ۸ گیگ آماده کن (ترجیحاً USB 3.0 برای سرعت بیشتر). فایل ISO پارچ لینوکس رو از [سایت رسمی پارچ](https://parchlinux.com/download) دانلود کن – نسخه‌های مختلف مثل Plasma یا GNOME رو انتخاب کن. بعد، با نرم‌افزارهایی مثل **Rufus** (برای ویندوز، رایگان و ساده) یا **Ventoy** (که اجازه می‌ده چند ISO رو روی یک فلش بذاری) فلش رو بوتیبل کن. در Rufus، ISO رو انتخاب کن و گزینه DD Image رو بزن.

کامپیوتر رو ری‌استارت کن، وارد BIOS/UEFI شو (معمولاً با کلیدهای F2, Del یا F10) و اولویت بوت رو به USB بده. از روی فلش بوت شو و وارد محیط زنده (Live) می‌شی. اینجا می‌تونی دسکتاپ پلاسما رو امتحان کنی، مرورگر باز کنی، فایل‌ها رو ببین، یا حتی VLC اجرا کنی و ویدیو پخش کنی. محیط زنده کاملاً عملی هست و می‌تونی اینترنت وصل شی و جستجو کنی.

تست‌های ضروری رو انجام بده:

- وای‌فای → آیا لیست شبکه‌ها رو نشون می‌ده و وصل می‌شه؟ اگر مشکلی بود، ممکنه نیاز به درایور اضافی باشه – در محیط زنده سرچ کن "linux wifi driver [مدل کارت]".
- صدا → با پخش موسیقی یا ویدیو بررسی کن. اگر صدا نداشت، چک کن آیا خروجی صدا درست انتخاب شده (در تنظیمات صدا).
- پرینتر → پرینتر رو وصل کن و یک صفحه آزمایشی چاپ کن. اگر کار نکرد، در CUPS (از طریق مرورگر localhost:631) تنظیم کن.
- وب‌کم و بلوتوث → اگر لپ‌تاپ داری، با Cheese (نرم‌افزار وب‌کم پیش‌فرض) تست کن. برای بلوتوث، دستگاهی مثل هدفون رو جفت کن.
- عملکرد کلی → برنامه‌ها رو باز کن، چند پنجره همزمان داشته باش و ببین آیا سیستم روان کار می‌کنه.

اگر مشکلی دیدی، در همین حالت زنده می‌تونی در اینترنت سرچ کنی و راه‌حل پیدا کنی، چون محیط زنده اینترنت داره.

گزینه دوم، نصب در ماشین مجازی مثل VirtualBox (رایگان از oracle.com) یا VMware هست. ISO رو در ماشین مجازی لود کن و پارچ رو کامل نصب کن. این روش ایمن‌تره چون هیچ تغییری در سیستم اصلی نمی‌ده و می‌تونی سخت‌افزار مجازی رو تنظیم کنی (مثل اختصاص رم بیشتر).

> **نکته:** اگر سیستم قدیمی داری و دیدی پلاسما سنگین عمل می‌کنه (مثلاً روی CPU قدیمی یا رم کم)، می‌تونی از حالت Safe Mode در بوت استفاده کنی یا در ماشین مجازی رم و سی‌پی‌یو رو افزایش بدی. همچنین، نسخه GNOME رو امتحان کن که سبک‌تره. اگر گرافیک مشکلی داشت، گزینه nomodeset رو در بوت اضافه کن.  
{.is-info}

## گام سوم: آماده‌سازی برای نصب نهایی

اینجا باید تصمیم بگیری: فقط پارچ نصب کنی (و ویندوز رو حذف کنی) یا دوگانه‌بوت (ویندوز + پارچ). برای تازه‌کارها، **دوگانه بوت** توصیه می‌شه چون می‌تونی هر دو سیستم رو داشته باشی و آرام به لینوکس عادت کنی.

- در ویندوز، وارد **Disk Management** شو (با جستجو در استارت) و از درایو C یا دیسک اصلیت، حداقل ۵۰ تا ۱۰۰ گیگ فضای خالی درست کن (با Shrink Volume). اگر فضای کمی داری، فایل‌های اضافی مثل Temp یا Recycle Bin رو پاک کن.
- سیستم رو Defrag کن (با ابزار Defragment and Optimize Drives) تا پارتیشن‌ها مرتب بشن.
- اینترنت پایدار داشته باش (ترجیحاً کابلی، چون وای‌فای ممکنه در نصب قطع بشه). اگر ADSL داری، مطمئن شو که مودم تنظیمه.
- اگر SSD داری، چک کن آیا TRIM فعاله (در ویندوز معمولاً هست). اگر ویندوزت با **UEFI** نصبه (چک کن در System Information)، به پارتیشن EFI (معمولاً ۱۰۰-۵۰۰ مگ fat32) دقت کن – نباید حذف بشه، چون GRUB ازش استفاده می‌کنه.
- فایل‌های مهم رو دوباره بک‌آپ بگیر و چک کن آیا همه چیز آماده است.

> **نکته:** اگر روی لپ‌تاپ رمزگذاری BitLocker فعاله (چک کن در Control Panel > BitLocker Drive Encryption)، قبل از نصب لینوکس حتماً غیرفعالش کن و کلید recovery رو ذخیره کن. در غیر این صورت، ممکنه دسترسی به ویندوزت رو از دست بدی. همچنین، اگر Secure Boot فعاله، در BIOS غیرفعالش کن تا نصب راحت‌تر بشه.  
{.is-warning}

## گام چهارم: نصب و تنظیم پارچ لینوکس

حالا زمان نصب اصلی رسیده. این گام رو با دقت دنبال کن تا مشکلی پیش نیاد.

۱. فلش بوتیبل رو وصل کن و سیستم رو ری‌استارت کن. وارد بوت منو شو و USB رو انتخاب کن.  
۲. در نصب‌کننده Calamares، زبان (فارسی اگر می‌خوای)، کیبورد (Persian) و منطقه زمانی (Asia/Tehran) رو انتخاب کن.  
۳. برای پارتیشن‌ها (مهم‌ترین بخش):  
   - فضای خالی رو انتخاب کن و پارتیشن‌های جدید بساز.  
   - `/` (ریشه) با فرمت ext4 و حداقل ۲۰ گیگ – اینجا سیستم‌عامل نصب می‌شه.  
   - `/boot/efi` با فرمت fat32 (اگر UEFI داری، معمولاً موجوده – اگر نه، ۵۰۰ مگ بساز).  
   - `swap` در صورت نیاز (معادل رم، مثلاً ۸ گیگ اگر رم ۸ گیگ داری) برای hibernation.  
   - `/home` برای فایل‌های شخصی با بقیه فضا – این کار داده‌ها رو جدا نگه می‌داره.  
   اگر مطمئن نیستی، گزینه Automatic Partitioning رو انتخاب کن اما دستی بهتره برای کنترل بیشتر.

۴. حساب کاربری بساز (نام کاربری، رمز عبور قوی) و اگر خواستی root رو فعال کن.  
۵. نصب رو شروع کن و منتظر بمون (۱۰-۳۰ دقیقه بسته به سیستم).  

بعد از ری‌استارت، منوی بوت GRUB ظاهر می‌شه که می‌تونی ویندوز یا پارچ رو انتخاب کنی. وارد پارچ شو. حالا وقت نصب نرم‌افزارهاست:

ابتدا سیستم رو به‌روز کن:

```bash
sudo pacman -Syu
```

سپس نرم‌افزارهای ضروری رو نصب کن:

```bash
sudo pacman -S libreoffice-fresh vlc firefox gimp
```

اگر خواستی از مخزن کاربران (AUR) نصب کنی (برای بسته‌هایی که در مخازن رسمی نیستن)، ابزار **paru** رو فعال کن:

```bash
git clone https://aur.archlinux.org/paru.git
cd paru
makepkg -si
paru -S نام_بسته  # مثلاً paru -S google-chrome
```

> **نکته:** نصب‌کننده گرافیکی **Calamares** ساده‌ست و مراحل رو قدم‌به‌قدم نشون می‌ده، اما اگر خطایی دیدی (مثل مشکل پارتیشن)، لاگ‌ها رو در /var/log/calamares.log بخون و بعد در انجمن پارچ بپرس. اگر نصب شکست خورد، از محیط زنده دوباره امتحان کن.

## گام پنجم: نگهداری و استفاده بلندمدت

حالا سیستمت آماده است، ولی نگهداری خیلی مهمه تا سیستم همیشه روان و امن بمونه. پارچ یک توزیع rolling release هست، یعنی به‌جای نسخه‌های بزرگ، همیشه آپدیت‌های کوچک می‌گیری.

### به‌روزرسانی منظم

حداقل هفته‌ای یک بار سیستم رو به‌روز کن:

```bash
sudo pacman -Syu  # برای بسته‌های رسمی
paru -Syu         # اگر AUR داری
```

### پاکسازی بسته‌های اضافی و کش

برای جلوگیری از پر شدن فضا:

```bash
sudo pacman -Rns $(pacman -Qtdq)  # حذف بسته‌های یتیم
sudo pacman -Scc                  # پاک کردن کش
```

### پشتیبان‌گیری منظم

از فایل‌های مهم بک‌آپ بگیر، مثلاً هفتگی:

```bash
rsync -a /home/ /backup/  # کپی فایل‌های home به درایو بک‌آپ
```

می‌تونی از ابزارهای گرافیکی مثل Timeshift برای اسنپ‌شات سیستم استفاده کنی.

### امنیت پایه

فایروال رو فعال کن:

```bash
sudo ufw enable
```

آنتی‌ویروس نصب کن (هرچند لینوکس کمتر نیاز داره):

```bash
sudo pacman -S clamav
sudo freshclam  # به‌روز کردن دیتابیس
```

رمز عبور قوی داشته باش و از sudo با دقت استفاده کن.

### دستورات پایه برای شروع

* `ls` → لیست فایل‌ها در دایرکتوری فعلی (با `ls -l` جزئیات بیشتر).
* `cd مسیر` → تغییر مسیر (مثل cd /home/user).
* `man دستور` → راهنمای دستور (مثل man ls).
* `pwd` → نمایش مسیر فعلی.
* `mkdir نام` → ساخت پوشه جدید.
* `rm فایل` → حذف فایل (با احتیاط!).

> **نکته:** پارچ لینوکس یک توزیع rolling است. یعنی همیشه جدیدترین بسته‌ها رو می‌گیری، اما این یعنی گاهی آپدیت‌ها ممکنه تغییرات بزرگی بیارن و سیستم رو بشکنن. همیشه قبل از آپدیت بزرگ، نسخه پشتیبان کامل داشته باش و اخبار آرچ/پارچ رو چک کن. اگر مشکلی پیش اومد، از recovery mode در GRUB استفاده کن.
