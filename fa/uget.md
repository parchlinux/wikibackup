uget یک رابط گرافیکی برای مدیر بارگیری‌هایی مثل [wget](/fa/wget) و [aria2](/fa/aria2) است که به صورت نرم‌افزار آزاد منتشر شده است.

# نصب
برای نصب uget آن را از مخازن پارچ نصب کنید:

```bash
$ sudo pacman -S uget
```
