cp فرمانی برای کپی کردن پرونده یا دایرکتوری است.

## نمونه

### • کپی فایل

```
cp ~/file1 /tmp
```

فایل file1 از خانهٔ کاربر به دایرکتوری tmp/ کپی می‌کند.

### • کپی دایرکتوری

```
cp /opt ~
```

دایرکتوری opt/ را به خانهٔ کاربر کپی می‌کند.
